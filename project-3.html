Cricket Team Generator

Skills: React, Express, MySQL, CSS

Description: This is a web application that generates a random playing XI of cricketers through querying a MySQL database of cricketers, with three different modes to select from: Random, Traditional, and Advanced. Random generates a random and cogent team from the database, Traditional mode uses models that represent the structure of “traditional” cricket playing XI’s and uses one of a few selected models, and Advanced mode lets you build a customized XI in terms of roles. Once the team is generated, you can reorder the XI and copy it to your clipboard.

Inspiration: I’m an avid cricket fan, and participate in fantasy leagues and built this application to evaluate teams. This seemed like an intuitive idea for a project that uses a database and applies models, so I pursued it to gain hands-on full-stack experience.
